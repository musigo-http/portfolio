hashcrack

hashcrack is a project for the ethical hacking, with hashcrack you can make a brute force with incremental or wordlist mode in the hashed password

if you have the name of the hash is very simple to crack it with wordlist or incremental, but if you dont have the hash, hashcrack test the password with all hash that he know

i have integrate the verbosity option, this option is not by default because it slows down cracking

you can see the source code of musigo in github: https://github.com/musigo-http/hashcrack
